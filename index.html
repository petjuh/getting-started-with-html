HTML
A "language" used by computers and Internet.
Tags. Elements inside <..> symbols.

Basics
1. <.b> Bold tag --> Bold tag
2. <.em> Italic tag --> Italic tag
3. Many others -- can search online

Inline vs. Block
4. Inline: something without invisible "box" around it.
5. Block: Something w/ invisible "box" around it.
--Span and Div. Span is inline. Div is block.
--BR (break) tag is inline. P tag (paragraph?) is block.
